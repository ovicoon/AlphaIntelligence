AlphaIntelligence

AlphaIntelligence는 문자 단위 신호 기반 텍스트 생성 모델입니다.
딥러닝 프레임워크 없이 동작하며, 상태 유사도와 신호 감쇠를 기반으로 텍스트를 학습하고 생성합니다.
과거 입력과 상태를 메모리로 저장하고, 현재 상태와 비교해 가장 적합한 출력을 선택합니다.

**특징**

# 문자 단위 텍스트 학습 및 생성

# 입력 신호와 상태 신호 감쇠 기반 동작

# 과거 사례 기반 메모리 학습으로 환각 최소화

# CSV 대화 데이터 학습 가능

# 모델 저장 및 압축 지원(lzma, bz2, gzip)

**구성 요소**

InputLayer: 입력 문자를 처리하고 신호를 출력으로 전달

OutputLayer: 출력 후보 문자를 저장하고 신호를 수신

StateLayer: 현재 문맥과 상태 정보를 유지

Connector: 입력과 출력을 연결하며 과거 신호와 상태를 기반으로 가중치를 계산

학습 방식

학습은 질문-응답 쌍을 기반으로 진행됩니다.

질문 문자를 입력 레이어에 자극

정답 문자를 한 글자씩 처리

각 입력 문자에서 Connector를 통해 정답 문자의 신호와 상태 스냅샷을 저장

입력과 상태의 유사도를 기반으로 출력 후보 학습

이 과정은 전통적인 신경망 학습 방식이 아닌 사례 기반 메모리 축적 방식입니다.

텍스트 생성 방식

질문 문자를 자극

입력 레이어가 출력 레이어로 신호 전달

Connector가 과거 신호와 상태를 비교하여 유사도 기반 가중치 적용

가장 강한 신호를 가진 문자를 선택하여 출력

선택된 문자를 상태에 반영하고 다음 문자 선택 반복

종료 조건

<END> 토큰 도달

신호가 임계값 이하

반복 출력 감지

CLI 사용법

learn "질문" "답변" : 단일 Q/A 학습

learncsv <파일> [개수] : CSV 파일에서 학습

stimulate "질문" : 모델에 자극을 주고 출력 생성

save <파일> [압축] : 모델 저장 (압축 옵션: lzma, bz2, gzip, none)

load <파일> : 저장된 모델 불러오기

reset : 모든 신호 초기화

stats : 모델 통계 확인

help : 도움말 표시

exit : 프로그램 종료

CSV 형식

dialog 컬럼에 따옴표로 감싼 순차 대화 문장 포함

예시: "Hello" "Hi" "How are you?" "Good"

주요 설정

DECAY_RATE: 입력/출력 신호 감쇠율

STATE_DECAY_RATE: 상태 신호 감쇠율

SIMILARITY_K: 유사도 계산 기준

STATE_INFLUENCE: 상태 영향력

INITIAL_SIGNAL_STRENGTH: 초기 자극 신호

설정 값 변경으로 출력 다양성과 반복 제어가 가능합니다.

모델 저장/로드

지원 형식: lzma, bz2, gzip, none

압축된 파일로 저장 가능

저장 시 문자 수와 전체 Connector 히스토리 포함

제한 사항

문자 단위 모델로 장기 문맥 유지 어려움

반복 출력 가능

데이터가 늘어나면 메모리 사용량 증가

확률 기반 샘플링 미지원

용도

신경망 없이 텍스트 생성 실험

사례 기반 언어 모델 실험

연구, 학습, 교육용

게임 NPC 등 행동 모델로 사용
